Troubleshooting Steps:

1. Basic Actions
  • Try refreshing the page (F5 or Ctrl+R)
  • Go back to the previous page
  • Return to the dashboard
2. Clear Browser Data
  • Clear cache: Ctrl+Shift+Delete (Chrome/Firefox) or Cmd+Shift+Delete (Mac)
  • Clear cookies: Remove reNgine-ng cookies from browser settings
  • Clear local storage: Developer Tools → Application → Local Storage → Clear
  • Hard refresh: Ctrl+Shift+R (Chrome/Firefox) or Cmd+Shift+R (Mac)
3. Browser Issues
  • Try a different browser (Chrome, Firefox, Safari, Edge)
  • Disable browser extensions temporarily
  • Try incognito/private browsing mode
  • Check if JavaScript is enabled
4. Network & Connection
  • Check your internet connection
  • Try accessing from a different network
  • Disable VPN or proxy if used
  • Check firewall settings
5. Still Having Issues?
  • Contact us on Discord channel #support, check wiki for link
  • Check if other users are experiencing the same issue
  • Try again in a few minutes (server might be restarting)
